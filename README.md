# 212 宿舍倒垃圾自动化提醒系统

###  项目初衷
我们宿舍（212）实行每天一人倒垃圾的轮流制度。但现实中，大家总是会陷入“今天该谁了？”、“昨天是谁丢的？”、“我是不是轮到了？”的终极哲学思考。

为了彻底解决这个困扰，我用 Python 开发了这套自动化系统。它不仅能在线显示排班，还能每天准时在微信里“催债”。

---

###  核心功能
* **实时网页查询**：基于 Streamlit 部署的专属网页，随时随地查看当前及未来的排班。
* **微信自动通知**：利用 GitHub Actions 每天早上 7:05 定时触发，通过 PushPlus 接口将提醒推送到全宿舍人的微信上。
* **全自动化运行**：云端部署，无需本地电脑运行，彻底解放双手（和大脑）。

---

###  技术栈
* **语言**：Python 3.9
* **前端展示**：Streamlit (Web App)
* **自动化脚本**：GitHub Actions
* **消息推送**：PushPlus (WeChat Webhook)

---

###  运行逻辑
系统以 **2026年4月13日** 为锚点日期，由其中一名舍友开始轮班，程序会自动计算当前日期与初始日期的天数差，并按宿舍人员名单取模循环，确保公平公正公开。
