# 🗑️ 212 宿舍倒垃圾自动化提醒系统 v3.1.2

### 💡 项目初衷
我们宿舍（212）实行每天一人倒垃圾的轮流制度。为了彻底解决“今天该谁了？”的终极哲学思考，我将系统一路进化到了 **3.1.2 全栈旗舰版**。它不仅能在线显示排班，准时在微信里“催债”，如今还学会了精准放假、排版美化，甚至**驯服了跨国服务器的时区差**。

---

### 🚀 v3.1.2 重大更新：征服“时区幽灵”与全方位细节打磨
在系统走向成熟的过程中，我们不断进行深度优化：
1. **时区绝对同步 (v3.1.2 新增)**：修复了 GitHub Actions 海外服务器按 UTC 时间运行导致的“日期倒流”Bug。通过底层注入 `timedelta(hours=8)`，强制全系统以**北京时间（UTC+8）**为唯一基准，确保微信推送名单与网页端实时展示 100% 严丝合缝。
2. **精准节假日识别 (v3.1.1)**：修复了原版会将“普通周末”误判为假期的缺陷。现在系统仅在法定大长假（如国庆、春节）休眠，普通周末照常提醒，确保宿舍卫生不留死角。
3. **UI 像素级优化 (v3.1.1)**：注入自定义 CSS 魔法，解决长标题在网页端换行的尴尬，实现 PC 端单行完美对齐，并自动适配手机端显示。

这已经是一个高度成熟、具备人文关怀、极致追求 UI 体验且具备国际化时区处理能力的微型 SaaS 系统。

---

### ✨ 核心功能
* **🔐 网页管理控制台**：新增 Admin Panel，支持输入管理口令解锁。支持在手机端实时修改舍友名单、调整初始日期与负责人。
* **🌍 全球化时区校准**：无论部署在地球哪个机房，核心算法强制锚定东八区时间，杜绝推送错乱。
* **🏖️ 精准休眠算法**：基于 `chinesecalendar` 库，智能识别“法定节假日”与“普通休息日”。只有大家确定回家的长假才会静默，普通双休依然是倒垃圾的日子。
* **🎨 响应式 UI 适配**：采用 CSS 注入技术，强制标题在不同尺寸屏幕下均能呈现最佳视觉比例。
* **🛡️ 安全加固**：通过参数化查询抵御 SQL 注入；管理后台预留安全扩展接口，确保数据稳如泰山。
* **⏰ 微信全端同步**：GitHub Actions 闹钟每天 **07:05** 准时拉取最新云端配置，自动计算排班并推送至全员微信。

### 🛠️ 技术栈
* **语言**：Python 3.9 (Full-Stack Logic)
* **数据库**：Supabase (Cloud PostgreSQL)
* **前端框架**：Streamlit (Interactive UI + CSS Injection)
* **自动化脚本**：GitHub Actions
* **通信协议**：PushPlus WeChat Webhook
* **算法依赖**：`chinesecalendar` (高精度中国历法库)、`datetime.timezone` (时区校正)

### 📖 快速部署指南
如果你是 314 宿舍或其他宿舍的兄弟，请按照以下步骤部署：
1. **Fork 本仓库**：点击右上角 Fork 按钮。
2. **依赖安装**：确保 `requirements.txt` 中包含 `chinesecalendar` 等所有核心组件。
3. **数据库配置**：在 [Supabase](https://supabase.com/) 创建 `dorm_rules` 表并填入初始排班锚点。
4. **环境变量 (Secrets)**：
   - **GitHub**：设置 `SUPABASE_URL`、`SUPABASE_KEY`、`PUSHPLUS_TOKEN`。
   - **Streamlit**：额外增加 `ADMIN_PASSWORD` 作为你的网页管理口令。
5. **修改宿舍号**：将代码中所有的 `dorm_id` 改为你自己的宿舍号。

---

### 🏗️ 运行逻辑
系统采用**双向数据流 + 动态逻辑校验**架构：
1. **时区与休眠网关**：闹钟触发后，强制转换北京时间，随后识别 `on_holiday` 与 `holiday_name`。若仅为周末则放行，若为法定大长假则静默。
2. **读取流**：网页与 GitHub 闹钟实时从新加坡数据库同步最新的排班规则。
3. **写入流**：管理员通过加密验证后，直接覆写云端数据库，实现排班规则的热修正。
4. **CSS 渲染层**：前端强制执行字体大小微调，确保各端视觉统一。
